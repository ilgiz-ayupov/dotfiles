# Конфигурационные файлы

## Fish

### Установка [Fish](https://fishshell.com/docs/current)
- #### Fedora
```shell
sudo dnf install fish
```
- #### Debian, Ubuntu, Mint
```shell
sudo apt install fish
```
- #### ALT Linux
```shell
epm -i fish
```

### Установка [Starship](https://github.com/starship/starship)
Минимальная, невероятно быстрая и бесконечно настраиваемая подсказка для любой оболочки!
```shell
curl -sS https://starship.rs/install.sh | sh
```

### Установить fish по умолчанию
1. Выполнить следующую комнду
```shell
chsh -s /usr/bin/fish
```
2. изменить оболочку для пользователя в файле /etc/passwd - поле shell


### Дополнительные программы 

#### [bat](https://github.com/sharkdp/bat)
bat - консольная утилита, которая является более усовершенствованным аналогом стандартной консольной утилиты cat. Bat имеет ряд преимуществ:
- Подсветка синтексиса
- Git интеграция
- Показывать непечатаемые символы

Команда установки:
- #### Fedora
```shell
sudo dnf install bat
```
- #### Debian, Ubuntu, Mint
```shell
sudo apt install bat
```
- #### ALT Linux
```shell
epm -i bat
```

#### [exa](https://github.com/ogham/exa)
exa - консольная утилита, которая является более усовершенствованным аналогом стандартной консольной утилиты cat. Exa имеет ряд преимуществ:
- Цвета для различения типов файлов и метаданных
- Git интеграция

Команда установки:
- #### Fedora
```shell
sudo dnf install bat
```
- #### Debian, Ubuntu, Mint
```shell
sudo apt install bat
```
- #### ALT Linux
```shell
epm -i bat
```
