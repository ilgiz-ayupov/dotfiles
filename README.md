# Конфигурационные файлы

## Fish

### Установка [Fish](https://fishshell.com/docs/current)
- #### ALT Linux
```shell
epm -i fish
```

### Установка [Starship](https://github.com/starship/starship)
Минимальная, невероятно быстрая и бесконечно настраиваемая подсказка для любой оболочки!
```shell
curl -sS https://starship.rs/install.sh | sh
```

### Установить fish по умолчанию
1. Выполнить следующую комнду
```shell
chsh -s /usr/bin/fish
```
2. изменить оболочку для пользователя в файле /etc/passwd - поле shell


### Дополнительные программы 

#### [bat](https://github.com/sharkdp/bat)
bat - консольная утилита, которая является более усовершенствованным аналогом стандартной консольной утилиты cat. Bat имеет ряд преимуществ:
- Подсветка синтексиса
- Git интеграция
- Показывать непечатаемые символы

Команда установки:
- #### ALT Linux
```shell
epm -i bat
```

#### [exa](https://github.com/ogham/exa)
exa - это современная замена устаревшей программы командной строки ls, поставляемой с операционными системами Unix и Linux, обладающая более широкими возможностями и лучшими настройками по умолчанию. Она использует цвета для различения типов файлов и метаданных. Она знает о симлинках, расширенных атрибутах и Git. При этом она маленькая, быстрая и состоит всего из одного двоичного файла.

Команда установки:
- #### ALT Linux
```shell
epm -i bat
```

#### [bottom](https://github.com/ClementTsang/bottom)
Настраиваемый кроссплатформенный графический монитор процессов/систем для терминала.
Поддерживает Linux, macOS и Windows. Создан на основе gtop, gotop и htop.

Команда установки:
- #### ALT Linux
```shell
epm -i bottom
```
